About "Space Cells"

This project was created for the ART75 class mashup project. The goal was to create an image which melded together two ideas, and in my case, I chose to represent space as a system of microscopic organisms. The project enmeshes planetary structures with cellular ones, showing cross-sections of cells out of textbooks melded with objects in space. I enjoyed the creation of this project a lot, though I lacked the technical skills to fully deveop it.
